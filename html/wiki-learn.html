Wikipedia Learn - MVP

"Transformer la curiosité en savoir structuré et gamifié."

React.js GSAP / Framer Motion Coding Sprint 48h

Présentation du projet

Développé lors d'un Coding Sprint de 48h à HETIC, Wikipedia Learn est une plateforme d'apprentissage qui structure l'immense base de données de Wikipedia. L'objectif est de sortir de la lecture passive pour entrer dans un parcours engageant.

Points clés

  • Parcours structurés : Thématiques découpées en étapes logiques.
  • Gamification : Gain d'expérience (XP) et déblocage de badges.
  • Interface immersive : UX pensée pour la concentration et le plaisir d'apprendre.

Détails techniques

Frontend : React.js (Vite)

Animations : Framer Motion & GSAP

Déploiement : Netlify
