Analyse des symptômes COVID-19

> Google Sheets | Analyse de Données | Santé

Contexte : objectifs et problématique

Ce projet analyse des données cliniques réelles pour comprendre comment les symptômes (toux, fièvre, etc.) varient selon le profil des patients. La problématique centrale était d'identifier les facteurs démographiques les plus corrélés aux diagnostics positifs pour mieux anticiper les besoins hospitaliers.

Rôle : votre contribution

J'ai agi en tant qu'analyste de données unique sur ce projet. Mes missions comprenaient :

  • Le nettoyage d'un dataset brut de données de santé.
  • La création de formules de calcul pour segmenter les patients.
  • La conception visuelle du rapport pour rendre les données exploitables.

Processus : étapes et défis

Le travail a été structuré en trois étapes clés :

  1. Nettoyage : Normalisation des données et traitement des valeurs manquantes via Google Sheets.
  2. Analyse : Utilisation de fonctions complexes comme NB.SI.ENS pour croiser les symptômes avec les tranches d'âge.
  3. Visualisation : Création de graphiques montrant les disparités géographiques et symptomatiques.
Défi : Gérer l'irrégularité des données saisies manuellement pour obtenir des statistiques fiables.

Résultats : impacts mesurables

L'analyse a permis de démontrer que la tranche d'âge 31-85 ans représentait environ 65% à 70% des cas positifs confirmés dans l'échantillon. Ces résultats confirment l'importance du ciblage démographique dans les politiques de santé publique.
